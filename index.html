✨ What's New — v1.5.3

🛡️ Data Protection Fixes
Saving your property settings no longer wipes your re-contact list or pencil expiry config. The midnight pencil timer now clears on sign out so it cannot multiply in the background.
📅 Extend Stay Fix
Extending a stay to a date where only a checkout occurs (turnaround day) no longer incorrectly blocks the extension.
✏️ Pencil Booking Fixes
Editing a pencil booking now checks for conflicts. Cancelling a pencil booking shows an inline confirm instead of a browser pop-up. Confirming a pencil with displaced holds correctly closes all modals. The Important Notices panel now runs an expiry check before display.
🗂️ Calendar & Data Fixes
Hover highlight classes now clear correctly when navigating months. Turnaround tiles on past dates are shown as muted. Repeat guest count now only counts confirmed bookings. Guest Masterlist lookups now use name + mobile for accuracy.
🔔 Notices Fix
Pencil expiry comparison now uses local time (not UTC midnight), matching actual expiry logic. Recently expired archive filter uses timestamp instead of checkout date.
